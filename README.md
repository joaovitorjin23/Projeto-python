O **Analisador de Notas e Frequência Escolar** é um sistema desenvolvido em Python para cadastrar alunos, calcular a média ponderada das avaliações e determinar automaticamente a situação final de cada estudante com base nas notas e na frequência.

Este projeto foi desenvolvido com foco em prática de lógica de programação, validação de dados e uso de estruturas básicas da linguagem Python.

## 🚀 Funcionalidades

- ✅ Cadastro de alunos.
- ✅ Validação de notas entre 0 e 10.
- ✅ Cálculo de média ponderada.
- ✅ Verificação da frequência escolar.
- ✅ Classificação automática do aluno.
- ✅ Geração de relatório final no terminal.

## 🧮 Regras do sistema

A média é calculada com os seguintes pesos:

- **AV1** → peso 2
- **AV2** → peso 3
- **AV3** → peso 5

### Fórmula utilizada
```python
(media) = (n1 * 2 + n2 * 3 + n3 * 5) / 10
```

## 📌 Critérios de situação

- **Frequência menor que 75%** → Reprovado por Frequência
- **Média maior ou igual a 7** → Aprovado
- **Média maior ou igual a 5** → Recuperação
- **Média menor que 5** → Reprovado por Nota

## 🛠️ Tecnologias utilizadas

- Estruturas condicionais

- Listas
- Dicionários

## 📚 Objetivo

O objetivo do projeto é automatizar a análise de desempenho escolar, tornando o processo de cadastro, validação e classificação dos alunos mais rápido, organizado e eficiente.
